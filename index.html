Introducere in HTML

Ce este HTML?

HTML (HyperText Markup Language) este folosit pentru a crea pagini web. Folosește taguri și atribute HTML pentru a descrie structura și formatarea unei pagini web.
HTML este scris cu diferite elemente, care sunt responsabile pentru a spune motoarelor de căutare cum să afișeze conținutul paginii. De exemplu, titluri, liste, imagini, link-uri și multe altele.

HTML folosește taguri și elemente predefinite care instruiesc browser-ul cum să afișeze conținutul. Elementele HTML includ o etichetă de deschidere, un anumit conținut și o etichetă de închidere. Este important să rețineți că includeți etichete de închidere. Dacă este omis, browserul aplică efectul etichetei de deschidere până la sfârșitul paginii. Această secțiune se va scufunda în structura de bază a unei pagini HTML, care include elemente esențiale ale blocului de construcție, cum ar fi declarația doctype, HTML, elementele head, title și body.

Structura de bază a unei pagini HTML este prezentată mai jos. Conține elementele esențiale ale blocului de construcție (de exemplu, declarația doctype, HTML, elementele head, title și body) pe baza cărora sunt create toate paginile web.

  • <!DOCTYPE html> - Aceasta este declarația tipului de document (nu din punct de vedere tehnic o etichetă). Acesta declară un document ca fiind un document HTML. Declarația doctype nu face distincție între majuscule și minuscule.
  • <html> - Acesta se numește elementul rădăcină HTML. Toate celelalte elemente sunt conținute în el.
  • <head> - Eticheta head conține elementele „din culise” pentru o pagină web. Elementele din cap nu sunt vizibile pe partea din față a unei pagini web. Elementele HTML utilizate în interiorul elementului <head> includ:
    • <style> - Această etichetă HTML ne permite să inserăm stil în paginile noastre web și să le facem atractive pentru a le privi cu ajutorul CSS.
    • <title> - Titlul este ceea ce este afișat în partea de sus a browserului dvs. atunci când vizitați un site web și conține titlul paginii web pe care o vizualizați.
    • <link> - Eticheta „link” este folosită pentru a lega HTML, CSS și JavaScript. Se auto-închide.
  • <body> - Eticheta body este folosită pentru a include tot conținutul vizibil al unei pagini web. Cu alte cuvinte, conținutul corpului este ceea ce browserul va afișa pe front end.

Taguri si elemente populare in HTML

Elementele de bază ale HTML, inclusiv etichetele, atributele, elementele și structura documentului, toate acestea se reunesc pentru a forma o pagină web funcțională.

Un alt element popular este headingul. Headingurile pot fi de mai multe tipuri. Ele se diferentiaza prin numere.

Headingurile sunt de mai multe tipuri:

Cel mai mare heading si cel mai important este h1, si cel mai mic este h6.

Brake sau <br> este folosit pentru a introduce o singură întrerupere de linie și nu necesită o etichetă de închidere.

Spanul este un subelement, folosit pentru a selecta si manipula un cuvant o propozitie sau o fraza.

hr este un element folosit pentru a adauga o linie spatiu sub forma de linie <hr/>. Nici acesta nu are nevoie de tag de inchidere.

Sup ( <sup> ) definește textul superscript. Textul în indice apare cu o jumătate de caracter deasupra liniei normale și uneori este redat într-un font mai mic. Textul suprascript poate fi folosit pentru note de subsol, cum ar fi WWW[1].
Alte elemente de formatare folosite:

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Smaller text
  • <del> - Deleted text
  • <sub> - Subscript text

Toate informatiile despre HTML, sau cel putin o buna parte, pot fi gasite pe google.

Style este un tag folosit pentru stilizarea paginii. Acesta poate fi folosit inline in interiorul tag-ului, in Head sau importat dintr-un fisier separat de tip style.css.

Div este un block-element/container folosit pentru a grupa mai multe elemente impreuna, eg:

Red
Yellow
Blue

Section este un tag ce defineste o sectiune intr-un document.

Nav tag defineste un set de link-uri navigabile. Nu toate link-urile unui document trebuie sa fie intr-un element de tip <nav>. Acest element este folosit in mare parte pentru blocuri majore de link-uri pentru navigatie.

Entities sunt niste grupari de litere, semne si numere care deservesc la proiectarea in pagina a caracterelor speciale de tip ©, ®, &...
Atunci cand dorim sa afisam aceste HTML Character Entities asa cum sunt ele scrise, putem folosi face un breakdown al gruparii (£) &pound; folosind pentru &, & + amp; urmat de restul caracterelor pound;.

Comentariile HTML sunt incluse în <!– și –> și servesc la adnotarea codului pentru dezvoltatori. Acestea oferă explicații, mementouri sau instrucțiuni care sunt invizibile pe pagina web redată. Aceste comentarii ajută la colaborare și îmbunătățesc lizibilitatea codului, oferind perspective asupra structurii și scopului codului.

Atributele HTML

Atributele HTML oferă informații suplimentare despre elemente. Specificate în eticheta de început, atributele vin de obicei în perechi nume/valoare, cum ar fi mane=”value”. Atributele definesc proprietățile și comportamentul, influențând afișarea conținutului și interacțiunea pe paginile web.

Cele mai folosite atribute sunt:

  • id - un identificator unic pentru un element HTML, folosit pentru stil sau interacțiune JavaScript. (ID="ID_name")
  • class - definește unul sau mai multe nume de clasă pentru un element, utilizate pentru stilizarea și aplicarea regulilor CSS. (class="Class_name")
  • src - specifică adresa URL sursă pentru resurse externe, cum ar fi imagini, audio sau video. (src="link-ul_sursa")
  • href - specifică adresa URL a resursei legate, utilizată de obicei în elementele ancora (a) pentru hyperlinkuri. (<a href="link">Text descriptiv</a>)
  • alt - oferă text alternativ pentru imagini, afișat dacă imaginea nu poate fi încărcată sau nu este disponibilă. (alt="Text_descriptiv")
  • width si height, sunt atribute care ofera detalii despre dimensiunea imaginii.
  • style este un atribut care atribuie multiple optiuni de stilizare unei linii de cod in html, aceasta optiune numita si in-line styling. Cateva dintre acestea pot fi: background-color, color, font-family, font-size, text-align. (<tagname style="property:value;">

Sintaxa: <element attribute_name= "attribute_value">
