JavaScript

Informații generale

JavaScript este un limbaj de scriptare sau de programare care vă permite să implementați caracteristici complexe pe paginile web - de fiecare dată când o pagină web face mai mult decât să stea acolo și să afișeze informații statice pe care să le vedeți - afișând actualizări de conținut în timp util, hărți interactive, animație 2D/ Grafică 3D, tonomat video cu defilare etc. — puteți paria că JavaScript este probabil implicat. Este al treilea strat al straturilor de tehnologii web standard, celelalte două fiind ( HTML și CSS ).

CSS

CSS este prescurtarea de la Cascading Style Sheets (sau „foi de stil in cascada”). Este un limbaj de reguli de stil pe care îl folosim pentru a aplica stilul conținutului nostru HTML, de exemplu, setarea culorilor și fonturilor de fundal și aranjarea conținutului nostru în mai multe coloane

HTML

HyperText Markup Language (HTML) limbajul de marcare pe care îl folosim pentru a structura și a da sens conținutului nostru web, de exemplu definirea paragrafelor, titlurilor și tabelelor de date sau încorporarea imaginilor și videoclipurilor în pagină.

JavaScript

JavaScript este un limbaj de scripting interpretat care vă permite să creați conținut care se actualizează dinamic, să controlați conținut multimedia, să animați imagini și aproape orice altceva. (Bine, nu totul, dar este uimitor ce poți realiza cu câteva rânduri de cod JavaScript.)

#1 Curs JS

Inserarea scripturilor JavaScript


  1. • Metoda 1 – plasarea script-ului în antet-ul paginii (între <head> şi </head>).
    Scripturile care trebuie executate cînd sunt apelate sau cînd are loc un eveniment, trebuie scrise în secţiunea head. În acest fel, scriptul va fi sigur încărcat înainte de a fi utilizat.
  2. • Metoda 2 – plasarea script-ului în corpul paginii (între <body> şi </body>). Scripturile care trebuie executate cînd pagina se incarcă trebuie scrise în secţiunea body şi vor genera conţinutul paginii.
  3. • Metoda 3 – utilizarea fişierelor sursă externe. acă dorim să utilizăm acelaşi script în mai multe pagini web fără a rescrie codul, trebuie să scriem scriptul JavaScript într-un fişier extern. Fişierul trebuie să aibă extensia .js şi nu poate conţine tagul <script>. Pentru a utiliza fişierul extern, indicăm numele fişierului în atributul src al tagului <script>.
  4. • Metoda 4 – crearea unui gestionar de evenimente. Nu este obligatoriu ca toate script-urile JavaScript să se găsească în interiorul tag-urilor <script> ... </script>. Puteţi apela, de asemenea la script-uri sub forma gestionarilor de evenimente, care indică navigatorului cum să reacţioneze atunci când se produc anumite evenimente. (<p onclick="alert('Imi place sa fiu asa cum sunt')"> Apasa aici!</p>)

Tipuri de date


În JavaScript există 8 tipuri de date primitive diferite care pot conține valori:

Number:

  1. INTEGER: În JavaScript, un întreg este un număr întreg fără o parte fracțională. Poate fi un număr pozitiv, un număr negativ sau zero.
  2. FLOATING POINT : Un număr în virgulă flotantă este în baza 10. El poate fi pozitiv sau negativ şi poate conţine zecimale.

String:

  1. Un şir de caractere (string, în limba engleză) este compus din litere, cifre, simboluri, caractere speciale şi secvenţe de ieşire. Conţinutul unui şir de caractere este considerat tot timpul ca fiind text, chiar dacă el constă din cifre şi simboluri numerice. Un şir de caractere este încadrat de ghilimele simple sau duble. Un şir de caractere încadrat de ghilimele duble poate fi inclus într-un şir încadrat de ghilimele simple şi vice versa.

Boolean:

  1. Valorile boleene sunt în număr de două: true (adevărat) şi false (fals). Cele două valori se folosesc pentru a indica dacă rezultatul evaluării unei condiţii este adevărat sau nu.

Undefined:

  1. Este o valoare primitivă atribuită automat variabilelor care tocmai au fost declarate sau argumentelor formale pentru care nu există argumente reale.

Null:

  1. Valoare ce reprezintă o referință care indică, în general, în mod intenționat, către un obiect sau o adresă inexistentă sau nevalidă. Semnificația unei referințe nule variază între implementările limbajului.
