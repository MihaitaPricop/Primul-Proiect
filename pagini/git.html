Despre Git

Git este un sistem de control al versiunilor care urmărește inteligent modificările din fișiere. Git este deosebit de util atunci când tu și un grup de persoane faceți modificări aceleași fișiere în același timp. De obicei, pentru a face acest lucru într-un flux de lucru bazat pe Git, ați:

  • Creați o ramificare din copia principală a fișierelor la care lucrați dvs. (și colaboratorii dvs.).
  • Efectuați modificări ale fișierelor în mod independent și în siguranță pe propria ramură personală.
  • Permiteți Git să îmbine în mod inteligent modificările dvs. specifice înapoi în copia principală a fișierelor, astfel încât modificările dvs. să nu afecteze actualizările altor persoane.
  • Lăsați Git să țină evidența modificărilor dvs. și ale altor persoane, astfel încât să continuați să lucrați la cea mai recentă versiune a proiectului.

Dacă doriți să aflați mai multe despre Git, consultați „ Despre Git ”.
